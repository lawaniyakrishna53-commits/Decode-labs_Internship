# Objective

- Build a simple AI Recommendation System that suggests items based on a user's preferences. The project focuses on using logic and similarity matching, not machine learning.

# Main Goal

Create a program that:

- Takes user interests or choices as input.
- Compares those interests with available items.
- Recommends the most suitable items.
# Key Requirements
- Accept user input (favorite category, hobby, etc.).
- Use simple if-else or similarity logic to compare preferences.
- Display personalized recommendations.

# Qualification Criteria
- Complete Project 3 successfully.
- Meet the required quality standards.
- This project is mandatory to unlock the next projects.
# Conclusion

- Adding a similarity score
- Allowing users to rate recommendations
- Experimenting with different recommendation methods
