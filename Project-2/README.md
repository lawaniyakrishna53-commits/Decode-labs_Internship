# Project Objective

The main goal is to build a machine learning classification model that can predict the category (class) of new data using previously learned examples.

# Project Requirements
Load a dataset.
Understand the dataset features.
Split data into training and testing sets.
Train a classification algorithm.
Test the model.
Measure the model's performance.

# Supervised Learning

Supervised learning is a machine learning technique where:

The model learns from labeled data.
Each input has a correct output.
After training, the model predicts labels for new data.
